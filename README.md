## Overview

Modern, responsive portfolio built with React and TypeScript, highlighting my expertise in Full-Stack development with strong backend and software architecture capabilities.

**Key Features:**
- Multilingual support (French, English, Italian)
- Dark/Light theme with user preference persistence
- Fully responsive design
- Optimized performance with Vite
- Clean, maintainable codebase

---

## Tech Stack

**Frontend:**
- React 18.3 with TypeScript 5.5
- Tailwind CSS for styling
- React Router for navigation
- react-i18next for internationalization
- Lucide React for icons

**Build & Tools:**
- Vite 5.4 (build tool)
- ESLint (code quality)
- Git & GitHub (version control)

---

## Project Architecture
```
src/
├── assets/          # Static resources and images
├── components/      # Reusable React components
│   ├── layout/     # Header, Footer
│   ├── sections/   # About, Projects, Skills, Experience, Contact
│   └── ui/         # Button, Card, Section, Container
├── context/        # React Context (Theme)
├── i18n/           # Internationalization configuration
│   └── locales/    # Translations (fr, en, it)
├── pages/          # Main pages (Home, NotFound)
└── App.tsx         # Root component
```
